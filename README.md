# Launch app DEV Mode

1. Install [Android Studio](https://developer.android.com/studio)
2. Install [Flutter](https://flutter.dev/docs/get-started/install)
3. Run `Flutter Doctor`
4. Run `Flutter pub get`
5. Run `flutter run`
